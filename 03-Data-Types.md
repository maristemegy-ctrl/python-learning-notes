# Data Types

Data types define the kind of value a variable stores.

---

## String (str)

Stores text.

Example:

name = "Mariam"
print(type(name))
Output:

<class 'str'>
---

## Integer (int)

Stores whole numbers.

Example:

age = 16
print(type(age))
Output:

<class 'int'>
---

## Float (float)

Stores decimal numbers.

Example:

height = 1.61
print(type(height))
Output:

<class 'float'>
---

## Boolean (bool)

Stores only two values:

- True
- False

Example:

student = True
print(type(student))
Output:

<class 'bool'>
---

## Check Data Type

Use type() to know the data type.

Example:

x = 25
print(type(x))
---

## Notes

- Text → str
- Whole numbers → int
- Decimal numbers → float
- True or False → bool
