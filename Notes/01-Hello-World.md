## What is Python?

Python is a high-level, easy-to-read programming language used in many fields such as Data Analysis, Artificial Intelligence, Web Development, and Bioinformatics.

---

## Your First Python Program

print("Hello, World!")
### Output

Hello, World!
---

## print()

The print() function is used to display text or values on the screen.

Example:

print("Welcome to Python")
print(2025)
Output:

Welcome to Python
2025
---

## Quotes

Python accepts both:

print("Hello")
and

``python
print('Hello')
``

---

## Comments

Comments are ignored by Python and are used to explain the code.

# This is a comment

## Notes
