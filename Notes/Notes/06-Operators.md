# Operators

## Arithmetic Operators

| Operator | Meaning | Example |
|----------|---------|---------|
| + | Addition | 5 + 3 |
| - | Subtraction | 5 - 3 |
| * | Multiplication | 5 * 3 |
| / | Division | 10 / 2 |
| // | Floor Division | 10 // 3 |
| % | Modulus | 10 % 3 |
|  | Power | 2  3 |

## Examples

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a // b)
print(a % b)
print(a ** b)
## Comparison Operators

==
!=
>
<
>=
<=
Example:

x = 10
y = 5

print(x > y)
print(x == y)
## Logical Operators

and
or
not
Example:

age = 18

print(age >= 18 and age <= 30)
## Notes

- / returns a float.
- // returns an integer (floor division).
- % returns the remainder.
- ** means power.
