# Data Types

## String (str)

Stores text.

python
name = "Mariam"
print(type(name))

**Output**

text
<class 'str'>

---

## Integer (int)

Stores whole numbers.

python
age = 16
print(type(age))

**Output**

text
<class 'int'>

---

## Float (float)

Stores decimal numbers.

python
height = 1.61
print(type(height))

**Output**

text
<class 'float'>
`

---

## Notes

- String → Text
- Integer → Whole numbers
- Float → Decimal numbers
